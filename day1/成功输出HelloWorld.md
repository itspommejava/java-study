## ✅ 一、HelloWorld 基本流程
### 1️⃣ 编写代码
文件名必须与类名一致，例如：**HelloWorld.java**

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("HelloWorld");
    }
}
```

### 2️⃣ 编译代码
```bash
javac HelloWorld.java
```

👉 成功后会生成一个 `HelloWorld.class` 文件（字节码文件）

### 3️⃣ 运行程序
```bash
java HelloWorld
```

输出：

```plain
HelloWorld
```

## ⚠️ 二、今天遇到的问题与解决方法
| 序号 | 问题表现 | 原因分析 | 解决方案 |
| --- | --- | --- | --- |
| 1️⃣ | `'QQ.EXE' 不是内部或外部命令` | 环境变量未正确保存 | 设置完路径后必须连续点击三次 “确定”，并重新打开 CMD |
| 2️⃣ | `找不到或无法加载主类 HelloWorld.java` | 用 `java`<br/> 直接执行 `.java`<br/> 文件 | Java 运行要先编译：`javac HelloWorld.java`<br/> → `java HelloWorld` |
| 3️⃣ | `找不到文件: HelloWorld.java` | 当前目录不对 或 文件名中有空格 | 使用 `dir`<br/> 查看文件；确认文件名中无空格或拼写错误 |
| 4️⃣ | `非法字符 '\uff08'`<br/>、`\u201c`<br/> 等 | 使用了中文括号、中文引号、中文分号 | 切换到英文输入法（半角）重新输入代码 |
| 5️⃣ | `找不到符号 printh` | 方法名拼写错误 | 正确方法应为 `System.out.println()` |
| 6️⃣ | 命令 `ｄ：`<br/> 无法执行 | 中文输入法状态下输入了全角字母 | 切换到英文输入法，输入半角 `d:`<br/> 即可切盘成功 |

## 💡 三、重点知识回顾
### 🔸 Java 文件规则
+ 文件名 **必须与 public 类名相同**
+ 文件扩展名为 `.java`
+ 编译后生成 `.class` 文件

### 🔸 命令行执行规则
| 操作 | 正确示例 | 错误示例 |
| --- | --- | --- |
| 切换盘符 | `D:` | `ｄ：` |
| 编译代码 | `javac HelloWorld.java` | `java HelloWorld.java` |
| 运行代码 | `java HelloWorld` | `java HelloWorld.class` |


### 🔸 英文/中文输入状态区别
| 类型 | 半角（英文） | 全角（中文） | 说明 |
| --- | --- | --- | --- |
| 括号 | `()` | `（）` | 全角括号会报错 |
| 引号 | `"` | `“”` | 全角引号无法识别 |
| 分号 | `;` | `；` | 全角分号非法字符 |
| 方括号 | `[]` | `【】` | 全角不被编译器识别 |


---

## 🧭 四、学习心得与注意事项
+ ✅ 写代码前，**切换到英文输入法（半角）**
+ ✅ 文件名、类名、方法名 **严格区分大小写**
+ ✅ 命令行中输入命令时，用 **英文符号和字母**
+ ✅ 每次修改环境变量后，要 **点击三次“确定”** 并 **重开 CMD**
+ ✅ 保持耐心，**每一个错误信息都是老师** ❤️

# 🧩 附录：Windows 命令行基础命令总结
💡 提示：命令行中的所有命令 **必须在英文输入法（半角状态）下输入**  
命令对大小写 **不敏感**（`CD` 和 `cd` 一样）

---

## 📂 一、目录与路径操作
| 命令 | 作用 | 示例 | 说明 |
| --- | --- | --- | --- |
| `dir` | 查看当前目录下的文件和文件夹 | `dir` | 类似于“查看文件夹内容” |
| `cd` | 进入指定目录（Change Directory） | `cd java学习` | 进入名为 `java学习`<br/> 的文件夹 |
| `cd ..` | 返回上一级目录 | `cd ..` | 两个点代表“上一层” |
| `cd \` | 返回根目录 | `cd \` | 回到当前盘符的根路径 |
| `d:` | 切换到 D 盘 | `d:` | 必须是英文状态的 `d:` |
| `c:` | 切换到 C 盘 | `c:` | 用来返回系统盘 |


---

## 🗂️ 二、文件与文件夹操作
| 命令 | 作用 | 示例 | 说明 |
| --- | --- | --- | --- |
| `md`<br/> 或 `mkdir` | 创建文件夹（Make Directory） | `md test` | 创建一个名为 test 的文件夹 |
| `rd`<br/> 或 `rmdir` | 删除文件夹 | `rd test` | 文件夹必须为空才能删除 |
| `del` | 删除文件 | `del HelloWorld.class` | 删除指定文件 |
| `copy` | 复制文件 | `copy a.txt b.txt` | 把 a.txt 复制为 b.txt |
| `move` | 移动/重命名文件 | `move a.txt D:\backup` | 移动到另一位置 |


---

## 🧹 三、屏幕与输出操作
| 命令 | 作用 | 示例 | 说明 |
| --- | --- | --- | --- |
| `cls` | 清屏（Clear Screen） | `cls` | 清空命令行的显示内容 |
| `exit` | 退出命令行 | `exit` | 关闭 CMD 窗口 |
| `echo` | 输出文字 | `echo Hello` | 在屏幕上显示 `Hello` |
| `pause` | 暂停执行 | `pause` | 显示“请按任意键继续...” |
| `type` | 查看文件内容 | `type HelloWorld.java` | 直接在 CMD 中显示文件内容 |


---

## ⚙️ 四、路径与环境变量
| 命令 | 作用 | 示例 | 说明 |
| --- | --- | --- | --- |
| `path` | 查看或设置系统 PATH 环境变量 | `path` | 查看当前生效的路径 |
| `echo %PATH%` | 查看 PATH 环境变量 | `echo %PATH%` | 查看完整路径列表 |
| `set` | 临时设置变量 | `set JAVA_HOME=D:\Java` | 仅在当前 CMD 有效 |
| `where` | 查找某个程序所在路径 | `where java` | 查找 `java.exe`<br/> 所在目录 |


---

## 💡 五、实用技巧与小提示
+ ✅ **命令不区分大小写**，`cd`、`CD`、`Cd` 都可以。
+ ✅ **Tab 自动补全**：输入文件夹/文件名的一部分后按 `Tab`，可自动补全。
+ ✅ **↑ ↓ 键**：查看历史命令。
+ ✅ **Ctrl + C**：强制中断正在运行的程序。
+ ✅ **右键粘贴**：在 CMD 里可以直接粘贴复制的命令。
